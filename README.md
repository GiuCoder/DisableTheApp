# DisableTheApp

# Download the script
```
 Invoke-WebRequest -Uri "https://raw.githubusercontent.com/GiuCoder/DisableTheApp/main/disableit.ps1" -OutFile "disableit.ps1"
```


# Run the script
```
./disableit.ps1
```
